ЗАВДАННЯ 5: КЛАСИФІКАЦІЯ ОЦІНОК
---------------------------------

МЕТА: Написати програму, яка класифікує оцінку студента (0-100) за категоріями: «відмінно», «добре», «задовільно» або «незадовільно».

---

1. АЛГОРИТМ (ОСНОВНІ КРОКИ):

* Введення: Запит оцінки (score) від користувача.
* Валідація: Перевірка, чи знаходиться оцінка в межах 0–100.
* Каскадна перевірка: Використовується структура if / else if / else для послідовної перевірки діапазонів, починаючи з найвищого (90) до найнижчого (0):
    - if (score >= 90)
    - else if (score >= 75)
    - else if (score >= 60)
    - else (решта)
* Виведення: Виводиться текстова назва оцінки.


---

2. СИНТАКСИС C++, ЩО ВИКОРИСТОВУЄТЬСЯ:

* Оператор if (умова): Умовний оператор. Початок перевірки.
* Конструкція else if (умова): Каскадна перевірка. Виконується, якщо всі попередні умови були хибними.
* Конструкція else: Блок за замовчуванням. Виконується, якщо жодна з попередніх умов не була істинною.
* Оператор >= (більше або дорівнює): Використовується для перевірки діапазонів.

---

3. ІНСТРУКЦІЯ ВИКОРИСТАННЯ:

* Компіляція: Перейдіть у каталог /Лабораторна1/задача5/ та виконайте g++ lab1_task5.cpp -o task5.
* Запуск: ./task5.
* Введення: Програма запросить одне ціле число (оцінку).

4. ПРИКЛАДИ ОЧІКУВАНИХ РЕЗУЛЬТАТІВ:
* Введення 95 → Відмінно
* Введення 80 → Добре
