# 2

## Завдання
Написати програму, яка виконує три послідовні дії:
1.  Запитує ціле число $X$ та виводить залишок при діленні його на 7.
2.  Запитує два цілих числа (A та B) і перевіряє, чи ділиться A на B без залишку. Якщо залишок є, його потрібно вказати.
3.  Перевіряє числа A та B на парність і виводить відповідне інформаційне повідомлення.

## Алгоритм 
1.  Введення X: Користувач вводить ціле число $X$.
2.  Обчислення залишку / 7: Використовується оператор `% 7`(оператор модуля) для знаходження залишку. Результат зберігається у змінній `zalishok`.
3.  Введення A та B: Користувач вводить два числа A та B.
4.  Перевірка подільності:Знаходиться залишок `A % B`. Застосовується умовна конструкція `if / else` для порівняння залишку з нулем.
5.  Перевірка парності: Для обох чисел A та B незалежно використовується перевірка `% 2`. Якщо результат `A % 2` дорівнює `0`, число вважається парним.

## Інструкція використання
1.  Компіляція: Перейдіть у каталог `/Лабораторна1/задача2/` та виконайте `g++ lab1_task2.cpp -o task2`.
2.  Запуск: `./task2`.
3.  Введення: Програма послідовно запросить три цілі числа.
